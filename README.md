# ML Playground

> A collection of notebooks, notes, exercises, and projects documenting my journey of learning Machine Learning from scratch.

This repository is organized according to my learning roadmap, starting with Python fundamentals and gradually progressing through data analysis, machine learning, deep learning, and real-world projects.

## 📖 About

The purpose of this repository is to:

- Build a strong foundation in Machine Learning.
- Learn by implementing concepts through code.
- Practice with real-world datasets.
- Document my learning progress.
- Maintain a well-organized collection of notebooks and projects.

Rather than only focusing on using libraries, I aim to understand the concepts and intuition behind Machine Learning algorithms.
